
# AI Agent Dev Team Boilerplate

Reusable boilerplate to build projects using AI-assisted development with Copilot or other agents.

## Structure

docs/
  AGENTS.md
  ARCHITECTURE.md
  ROADMAP.md

tasks/
  01-project-setup.md
  ...

NEXT_TASK.md

ai-dev-team/
  agents/
  graph/

## Workflow

1. Update NEXT_TASK.md
2. Ask Copilot to implement the task
3. Review generated code
4. Commit changes
5. Move to next task

Works with:
- GitHub Copilot
- LangGraph AI agents
- GitHub Issues workflows
