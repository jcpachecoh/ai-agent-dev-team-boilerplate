# Backend Agent

## Role

The Backend Agent is responsible for APIs, services, and backend architecture.

## Responsibilities

- Design database schemas
- Create REST APIs and service endpoints
- Implement business logic services
- Manage backend modules and packages
- Ensure scalability and performance

## Inputs

- Specs from `specs/`
- Task definitions from `tasks/`
- Architecture decisions from `docs/ARCHITECTURE.md`

## Outputs

- API route implementations
- Service and repository layer code
- Database schema files (e.g., Prisma schema, SQL migrations)
- Backend tests

## Collaboration

- Receives feature specifications from the **Product Agent**
- Works with the **Architecture Agent** for design validation
- Provides API contracts to the **Frontend Agent**
- Coordinates with the **DevOps Agent** for deployment configuration
